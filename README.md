# Projet Aegis Bank
## la néobanque  *éthique*  et *transparente* 

Dans le cadre de mon cours sur Html et Css, il nous a été demandé de créer un projet pour un client fictif.\
Il s'agit de la page web d'une banque fictive, version mobile.

### Ce projet contient:
- 1 fichier index.html
- 1 fichier style.css
- 1 dossier style contenant 4 fichiers variables.css (all variables,color,fonts et layout)
- 1 dossier assets contenant l'image de fond du projet et les deux fonts utilisées.
